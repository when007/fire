# FIRE 计划计算器

一个简洁美观的 FIRE (Financial Independence, Retire Early) 计划计算器，支持复利计算和可视化图表展示。

## 功能特点

- 📊 **复利计算**：准确计算年利息收入（不能为负，最少为0）
- 📈 **可视化图表**：使用 ECharts 展示逐年明细趋势
- 🎯 **灵活参数**：支持设置初始金额、年化利率、年支出、生存年数等参数
- 💡 **智能计算**：自动推导缺失参数
- 📱 **响应式设计**：适配各种屏幕尺寸

## 使用方法

1. 打开 `index.html` 文件
2. 填写相关参数（A. 初始现金、B. 年化利率、C. 年支出、D. 生存年数）
3. 点击"开始计算"按钮
4. 查看计算结果和逐年明细图表

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- ECharts 5.4.3

## 本地运行

使用 Python 内置服务器：

```bash
python3 -m http.server 8000
```

然后访问 `http://localhost:8000`

## 注意事项

- 年利息收入不会为负，最少为0（当余额为负或0时，利息为0）
- 所有金额显示不带人民币单位（¥符号）
- 图表支持交互式查看，鼠标悬停可查看详细数值
